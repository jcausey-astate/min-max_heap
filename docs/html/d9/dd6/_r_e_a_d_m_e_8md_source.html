1 ## Min-Max Heap
2 Defines functions for maintaining a Min-Max Heap, as described by Adkinson:
3 <blockquote>
4  M. D. Atkinson, J.-R. Sack, N. Santoro, and T. Strothotte. 1986.
5  Min-max heaps and generalized priority queues.
6  Commun. ACM 29, 10 (October 1986), 996-1000.
7  DOI=http://dx.doi.org/10.1145/6617.6621
8 </blockquote>
9 
10 The main advantage of the Min-Max Heap is the ability to access both the _minimum_ and _maximum_ values contained in the data structure in constant time. The trade-off is a slight increase in the complexity constant coefficients with respect to traditional heaps. The Min-Max heap still maintains the same _order_ of complexity as traditional heaps for all operations.
11 
12 The heap functions defined in _`mmheap.h`_ are defined as _templates_, so a heap of any type that is _less-than comparable_ and _copy-constructable_ is possible. The heap functions are designed to work in-place on top of a regular C++ array.
13 
14 ### Namespaces
15 The file _`mmheap.h`_ defines two namespaces: `mmheap` and `_mmheap`. All of the functions necessary to use the min-max heap are exposed in the `mmheap` namespace. It should not be necessary to use the `_mmheap` namespace (those functions are for internal use only).

17 ### Function Reference
18 Full reference documentation is available in the _"docs"_ project directory. Only the most commonly used functions are described here.
19 
20 #### `mmheap:: make_heap()`
21 `template <typename DataType>`<br />
22 `void make_heap (DataType∗ heap_array, size_t size);`<br />
23 Creates the min-max heap from an arbitrary C++ array, given the array and its size as arguments.
24 
25 #### `mmheap:: heap_insert()`
26 `template <typename DataType>`<br />
27 `void heap_insert (const DataType& value, DataType∗ heap_array, size_t& count, size_t max_size);`<br />
28 Inserts a new value into the heap, given the value, the heap array, the current number of items contained in the heap, and the maximum storage size of the array. The `count` will be increased by one following the function call.
29 
30 #### `mmheap:: heap_max()`
31 `template <typename DataType>`<br />
32 `DataType heap_max (DataType∗ heap_array, size_t count);`<br />
33 Returns the maximum value contained in the heap, given the heap array and the current number of items contained in the heap.
34 
35 #### `mmheap:: heap_min()`
36 `template <typename DataType>`<br />
37 `DataType heap_min (DataType∗ heap_array, size_t count);`<br />
38 Returns the minimum value contained in the heap, given the heap array and the current number of items contained in the heap.
39 
40 #### `mmheap:: heap_remove_max()`
41 `template <typename DataType>`<br />
42 `DataType heap_remove_max (DataType∗ heap_array, size_t& count);`<br />
43 Removes and returns the maximum value contained in the heap, given the heap array and the current number of items contained in the heap. The `count` will be decreased by one following the function call.
44 
45 #### `mmheap:: heap_remove_min()`
46 `template <typename DataType>`<br />
47 `DataType heap_remove_min (DataType∗ heap_array, size_t& count);`<br />
48 Removes and returns the minimum value contained in the heap, given the heap array and the current number of items contained in the heap. The `count` will be decreased by one following the function call.

50 #### Additional Functions
51 The following functions are less likely to be commonly used, but are provided under the `mmheap` namespace as well; for more information, _read the documentation in the `docs` directory_.
52 
53 ##### `mmheap:: heap_insert_circular()`
54 Add to heap, rotating the maximum value out if the heap is full.
55 
56 ##### `mmheap:: heap_replace_at_index()`
57 Replace the value at a specific index in the heap array with a new value, and restore the heap property.
58 
59 ##### `mmheap:: heap_remove_at_index()`
60 Remove and return the value at a specified index in the heap array, and restore the heap property.
61 
62 ##### `mmheap:: is_heap()`
63 Returns `true` if an arbitrary array is in a valid Min-Max heap ordering, or `false` otherwise.
